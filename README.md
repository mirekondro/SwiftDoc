# SwiftDoc

Minimal JavaFX app organized using MVC-style packages.

## Structure

- `src/main/java/dk/easv/swiftdoc/app` - JavaFX app startup
- `src/main/java/dk/easv/swiftdoc/controller` - UI command handlers
- `src/main/java/dk/easv/swiftdoc/model` - domain models
- `src/main/java/dk/easv/swiftdoc/service` - business logic/services
- `src/main/resources/dk/easv/swiftdoc/view` - FXML views

## Run

```zsh
./mvnw clean javafx:run
```
